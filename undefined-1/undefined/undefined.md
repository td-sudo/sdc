# 착신번호 등록

## 착신번호 등록하기

좌측 **운영관리 > 착신번호/계정관리 > 착신번호** 메뉴에서 착신번호를 등록해 주세요.

{% hint style="info" %}
등록한 번호는 ARS 시나리오에서 콜 연결 요청 시 해당 전화로 연결됩니다.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (25).png" alt=""><figcaption><p>착신번호 등록</p></figcaption></figure>

* 휴대폰번호를 등록하는 경우 별도의 통화료가 부과 됩니다. (14.5원/10초당)
* 전국 대표번호 요금제(베이직 플랜) : 착신번호 5개까지 무료로 등록할 수있습니다.(초과 등록 시 과금)
* 기존 번호 그대로 사용 요금제(Lite \~ Enterprise 플랜) : 착신번호 무제한 무료로 등록할 수 있습니다.
