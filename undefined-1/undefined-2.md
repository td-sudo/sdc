# 계정 등록

## 계정 추가 등록하기

좌측 **운영관리> 착신번호/계정관리** 메뉴에서 추가 계정을 등록하실 수 있습니다.

{% hint style="info" %}
계정을 추가하기 전에 대표번호와 연결되는 **착신번호**를 1개 이상 설정해 주세요.
{% endhint %}

계정 추가를 위해 '**계정 추가하기**' 버튼을 선택해 주세요.

<figure><img src="../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

상담을 진행할 상담사 **계정 정보**를 **입력**해 주세요.

<figure><img src="../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
상담사 권한의 계정은 로그인시 '**상담하기**' 기능만 사용 가능합니다.
{% endhint %}

<figure><img src="../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
사전에 등록한 착신번호별로 담당자를 지정하실 수 있습니다.
{% endhint %}

**계정 추가하기**에서 입력한 **이메일 주소**로 아래와 같은 가입 안내 메일이 발송됩니다.

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 12.31.02 PM (1).png>)

{% hint style="info" %}
수신한 이메일내에 **로그인하기** 버튼을 눌러 상담콜에 로그인 해주세요.
{% endhint %}

* 사용하실 **비밀번호**를 **설정**해 주세요.
* 비밀번호 설정을 건너뛰는 경우 우측 상단 **개인설정 메뉴**에서 변경 가능합니다.
