# 알림톡 템플릿

## 알림톡 템플릿 사용하기

좌측 **운영관리 > 알림톡 연동/템플릿** 선택시 페이지 하단에서 알림톡 템플릿 목록을 확인하실 수 있습니다.\
상담콜은 카카오 채널 아이디 연동 없이 사용 가능한 공용 프로필을 제공합니다.

{% hint style="info" %}
상담콜에서는 기본으로 공용 @arshappytalk 채널 아이디를 사용하여 알림톡을 발송하실 수 있습니다.
{% endhint %}

{% hint style="info" %}
비즈니스 인증을 완료한 채널아이디를 보유하고 계시다면, [**채널 연동 가이드**](undefined-1.md)를 참고하여 채널 아이디를 추가해 주세요.
{% endhint %}

알림톡으로 다양한 방식으로 고객 문의사항을 처리하실 수 있습니다.

🟡 빠른 **카톡 상담**으로 **전환**하기: 템플릿 목록 > 상담사연결요청, 버튼 타입 '**상담톡 전환**' 템플릿 사용 시 전화 문의를 채팅상담으로 전환하실 수 있습니다.

<figure><img src="../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>

🟢 **챗봇 호출**하기: 템플릿 목록 > 연결하기, 버튼 타입 '**봇키워드**' 템플릿을 사용하여\
챗봇 호출 문구를 '**연결하기**'로 설정하시면 버튼 선택 시 **챗봇**으로 **연결**시킬 수 있습니다.

🔵 **특정 URL로 이동**시키기: 템플릿 목록 > 연결하기, 버튼 타입 '**웹 링크 전달**' 템플릿을 사용하여 버튼 클릭시 특정 URL로 이동시킬 수 있습니다.

<figure><img src="../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

* 쇼핑몰의 **문의하기 게시판 URL**을 입력하여, 알림톡을 수신한 고객이 문의 게시판으로 유입되도록 설정하실 수 있습니다.
* 홈페이지의 **웹채팅 문의창 URL**을 입력하여, 알림톡을 수신한 고객이 웹채팅으로 전환 되도록 설정하실 수 있습니다.

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 5.17.13 PM.png>)

🟣 **근무 외 시간**에 활용하기: 템플릿 목록 > 문의하기, 버튼타입 '**메시지 전달**' 템플릿을 사용하면 고객의 문의사항을 카톡으로 접수하실 수 있습니다.

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 5.26.36 PM.png>)

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 5.27.59 PM.png>)

* **근무 외 시간** 또는 바로 응대가 힘든 상황에는 위와 같은 템플릿을 발송하여, **고객 문의를 카톡으로 접수**받고, 담당자가 **추후에 응대** 하도록 운영하실 수 있습니다.

{% hint style="danger" %}
**Multi Depth ARS를 이용하실 경우 Multi Depth Flow Builder - 메시지 발송 블럭**에서 템플릿을 선택하셔야 합니다.
{% endhint %}
