# 채널 아이디 연동

## 카카오 채널아이디 연동하기

좌측 **운영관리 > 알림톡 연동/템플릿** 메뉴에서 채널 아이디를 추가해 주세요.

<figure><img src="../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>

채널 아이디 연동을 위해 **비즈니스 인증**을 완료한 **카카오 채널아이디**를 준비해 주세요.

{% hint style="warning" %}
**채널 아이디** 입력시 반드시 앞에 '**@**' 를 포함하여 입력해 주세요.

**카카오 채널 아이디 (검색 아이디) 입력 예시**: @arshappytalk
{% endhint %}

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 2.13.23 PM.png>)

{% hint style="info" %}
채널 아이디 정보 확인을 위해 [**카카오 비즈니스**](https://accounts.kakao.com/login/kakaoforbusiness?continue=https://center-pf.kakao.com/) 페이지에 로그인 해주세요.
{% endhint %}

{% hint style="danger" %}
카카오 채널 아이디가 없으시다면 [**이곳에서**](https://accounts.kakao.com/weblogin/create_account?continue=https://center-pf.kakao.com/\&is_iframe=false\&lang=ko\&service=kakaoforbusiness) 새로운 채널아이디를 생성해 주세요.
{% endhint %}

* 아래 연동 가이드를 참고하여 **카카오 비즈니스 계정에 로그인 후** 필요한 세팅을 완료해 주세요.

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 4.38.55 PM.png>)

{% hint style="info" %}
좌측 메뉴 채팅 설정**에서** 1:1 채팅 사용 > **ON**으로 설정해 주세요.

좌측 메뉴 **관리** > **상세설정**에서 **채널홈 공개 설정**과 **검색 허용**을 모두 **ON**으로 설정해 주세요.
{% endhint %}

![검색용 아이디 확인](<../.gitbook/assets/Screen Shot 2021-07-19 at 4.39.11 PM.png>)

{% hint style="info" %}
좌측 메뉴 **관리** > **상세 설정**에 **검색용 아이디**를 확인, 채널 연동시 **@와 함께 입력**해 주세요.
{% endhint %}

![](<../.gitbook/assets/Screen Shot 2021-07-19 at 4.39.27 PM.png>)

{% hint style="info" %}
상단 메뉴 > 내 계정 설정 메뉴에서 휴대폰 번호를 확인해 주세요.

인증 번호 발송 요청시 해당 번호로 인증 코드가 전송됩니다.
{% endhint %}

{% hint style="success" %}
위에서 확인한 **검색아이디**와 등록하신 **카테고리**를 입력해 주세요.

관리자 휴대폰 전화번호 입력 후 인증 번호 전송 버튼을 선택해 주세요.

전송된 인증 코드를 입력하면 채널 연동이 완료됩니다!
{% endhint %}

* 채널 연동 작업 관련 문의 사항은 언제든지 [**해피톡 고객센터**](https://design.happytalkio.com/chatting?siteId=4000000015\&siteName=%ED%95%B4%ED%94%BC%ED%86%A1\&categoryId=61504\&divisionId=61505)로 문의 주세요.💙
