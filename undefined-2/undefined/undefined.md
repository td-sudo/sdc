# 일자별 상담콜

## 상담콜 일별 내역

좌측 **상담관리** **> 일자별 상담콜** 메뉴에서 대표 전화로 인입된 콜 현황을 일별로 확인하실 수 있습니다.

<figure><img src="../../.gitbook/assets/image (9).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
조회 기간을 설정할 수 있으며, ARS 시나리오 유형 선택, 콜 타입 / 통화상태 등을 선택하여 자세히 검색할 수 있습니다.
{% endhint %}

ARS 시나리오 유형은 'Single Depth ARS', 'Multi Depth ARS'를 선택할 수 있습니다. 현재 ARS에 설정되어 있는 유형이 디폴트로 검색됩니다.

<figure><img src="../../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
일별 내역은 엑셀 파일로 다운로드 가능합니다.
{% endhint %}
