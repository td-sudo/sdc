# 콜 이력

## 상담콜 이력

운영 중인 상담콜 이력을 확인하실 수 있습니다.

**✔️** [**Today 상담콜**](../undefined/today.md)**: 당일 기준**, 대표 전화로 인입된 **콜 현황**을 확인하실 수 있습니다.

**✔️** [**일자별 상담콜**](../undefined/undefined.md)**: 일자별**로 대표 전화로 인입된 **콜 현황**을 확인하실 수 있습니다.

✔️ [**콜 고객 리스트**](undefined-1.md): 대표 전화로 문의한 **고객 리스트**를 확인하실 수 있습니다.

✔️ [**알림톡 발송 현황**](undefined-6.md): 최근 발송한 알림톡 메시지 발송 현황을 확인하실 수 있습니다.
