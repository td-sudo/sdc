# 기존 전화번호로 사용 신청

현재 사용하고 계시는 전화번호에 ARS/IVR 기능을 도입할 수 있습니다.

*   상담콜에서 080 번호(수신자 부담)를 발급해 드리고, 해당 080 번호에 착신전환하여 이용하는

    방식으로 제공됩니다. (착신전환 비용 미발생)

{% hint style="info" %}
**‘기존 전화번호 그대로 가입할래요'**&#xB97C; 선택해 주세요.
{% endhint %}

<figure><img src="../.gitbook/assets/image (33).png" alt=""><figcaption></figcaption></figure>

* 원하시는 요금제를 선택해 주세요.
* ARS 처리되는 도수 기준으로 과금되는 방식입니다.\
  &#x20;(1도수 = 60초, 예 : 통화 1건 당 1초 \~ 60초 이내 발생 시 1도수, 61초\~120초까지 2도수로 처리)

<figure><img src="../.gitbook/assets/드림라인 요금제 안내_20250224.png" alt=""><figcaption><p>기존번호 그대로 사용하기 - 요금제</p></figcaption></figure>

* 상담콜 도입 신청서를 작성해 주세요.
* ‘현재 유선번호’ 항목에 지금 사용하고 계신 전화번호를 입력해 주세요.

<figure><img src="../.gitbook/assets/image (36).png" alt=""><figcaption><p>기존번호 그대로 사용하기 - 신청서 작성 화면</p></figcaption></figure>

모든 신청 과정이 완료되면 ‘기존 유선번호‘ 항목에 입력한 전화번호에서\
상담콜이 제공해 드리는 080 번호에 착신전환 설정하신 후 \[테스트 할게요]를 선택해 주세요.

<figure><img src="../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

상담콜에서 정상적으로 착신전환이 되었는지 확인 과정이 진행되며,\
이상 없이 착신전환이 성공한 경우 상담콜을 이용할 수 있습니다.\
※ 확인 과정은 최대 1분 정도 소요됩니다.
