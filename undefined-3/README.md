# 정식버전 도입하기

상담콜 정식버전 도입은상단 [**'정식버전 도입하기'**](https://ars.happytalk.io/app/join) 를 통해 간편하게 신청하실 수 있습니다.💙

<figure><img src="../.gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure>

*   상담콜은 현재 고객님이 사용중인 번호 그대로 이용하시면서 ARS/IVR 기능을 제공해 드리고

    있습니다.(일반 유선 전화번호, 전국 대표번호 그대로 이용 가능)
* 원하실 경우 8자리 전국 대표번호를 신규로 발급 받으실 수 있습니다.



{% hint style="info" %}
현재 보유하신 번호를 그대로 사용하면서 도입하시려면 **‘기존 전화번호 그대로 가입할래요'** 를 선택해 주세요.
{% endhint %}

![서비스 신청 타입 선택 - 기존 전화번호 그대로 가입할래요.](<../.gitbook/assets/image (32).png>)

* 02, 031 등 지역번호 기반의 일반 유선번호 / 070 인터넷전화 / 1688-xxxx 및 080번호 이용 시\
  도입 가능합니다.
* 정식버전 도입 후에는 상담사 자리번호(착신번호)를 추가 등록해야 하기 때문에\
  대표번호로 사용할 유선번호 1개와 상담사 자리번호(착신번호)용 전화번호 1개,\
  총 2개의 전화번호를 보유하신 경우 신청이 가능합니다.
* 상담사 자리번호(착신번호)는 휴대폰번호로도 등록할 수 있으며, 이 경우 휴대폰번호 착신비용이\
  추가로 발생합니다. (Lite 플랜은 무료)

{% hint style="info" %}
8자리 전국 대표번호를 새로 발급받고 싶으시면 **‘새로운 전국 대표번호(8자리) 발급 받을래요'**&#xB97C;\
선택해 주세요.
{% endhint %}

![서비스 신청 타입 선택 - 새로운 전국 대표번호(8자리) 발급 받을래요.](<../.gitbook/assets/image (34).png>)

* 8자리 전국 대표번호를 새로 발급받기 위해서는 추가 서류를 제출해 주셔야 합니다.
* 서류 처리 기간은 제출 이후 약 1\~2업무일이 소요 됩니다.
